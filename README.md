# Online-Store-Management-System
A Java-based online store management system supporting products, suppliers, discounts, customers, and order handling with admin and customer roles.

**Database, Online Store Management System**

This repository contains the database design for the Online Store Management System.

The schema is based on real functional requirements, including products, suppliers, customers, discounts, and order workflows.
